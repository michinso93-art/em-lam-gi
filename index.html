🚀 처음 설정 가이드

아래 순서대로 한 번만 설정하면 이후에는 완전 자동으로 동작합니다.

  1. 1
    Google Sheets 만들기sheets.new 접속 → 새 스프레드시트 생성
    URL에서 /d/와 /edit 사이의 ID를 복사
  2. 2
    Apps Script 설정스프레드시트에서 확장 프로그램 → Apps Script 클릭
    받은 apps_script.js 파일 내용을 전체 붙여넣기
  3. 3
    API 키 입력Apps Script 코드 상단 CONFIG에서
    SHEET_ID와 GEMINI_API_KEY를 본인 값으로 변경
  4. 4
    초기화 실행Apps Script에서 initializeSheets 함수 선택 후 실행
    권한 허용 팝업이 뜨면 모두 허용
  5. 5
    웹앱 배포Apps Script에서 배포 → 새 배포 → 웹앱
    액세스: 모든 사용자 선택 → 배포 URL을 아래 설정에 입력
  6. 6
    아래 설정 저장Sheets ID와 배포 URL 입력 후 저장
